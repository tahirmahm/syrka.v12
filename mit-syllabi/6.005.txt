MIT 6.005 — Elements of Software Construction
Source: MIT OpenCourseWare
URL: https://ocw.mit.edu/courses/6-005-elements-of-software-construction-fall-2011/

=== COURSE DESCRIPTION ===
This course introduces fundamental principles and techniques of software development. Students learn how to write software that is safe from bugs, easy to understand, and ready for change. Topics include specifications and invariants; testing, debugging, and version control; abstract data types, representation independence, and interfaces; design patterns for object-oriented programming; concurrent programming, including message passing and shared memory concurrency.

The course emphasizes writing correct, clear, and changeable code, with a strong focus on testing as a first-class engineering activity.
